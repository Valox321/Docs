# 🌌 Mentality Reborn UI Documentation

Welcome to the official documentation for the **Mentality Reborn UI Library**. This library is designed to be high-performance, visually stunning, and easy to integrate into your Roblox scripts.

### 🚀 Key Features
*   **Lucide Icons**: Support for over 1000+ icons by name.
*   **Performance Focused**: Optimized rendering and animations.
*   **Config Management**: Built-in integration with a professional SaveManager.
*   **Mobile Friendly**: Responsive design that works on all platforms.
