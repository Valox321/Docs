# Changelog

All notable changes to the UI Library will be documented in this file.

## [1.0.0] - 2026-05-16

### Added
- Initial release of the UI Library.
- Complete set of basic components: Buttons, Toggles, Sliders, Dropdowns, ColorPickers.
- Modern Dark Theme with glassmorphism effects.
- Support for Lucide Icons.
- Integrated SaveManager and ThemeManager.
- Comprehensive documentation site.
